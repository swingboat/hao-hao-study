📐 Session 装配流程（7 池 → 一次性 10-15 道题）

学生开始 25min session → 读 8 类输入信号 → 判断学生阶段 → 加载配比配置 → 各池按配额召回 → 去重 + 场景覆盖 → 难度排序 → 输出题目列表

🔄 端到端数据流（一次 session 全链路）

sequenceDiagram
    participant U as 学生
    participant R as Recommender
    participant DB as 数据层
    participant P as 7 池召回器
    participant Q as 题库
    U->>R: 开始 25min session
    R->>DB: 读 8 类信号
    DB-->>R: mastery/错题/到期/...
    R->>R: 判阶段 + 加配比
    R->>P: 按配比下达召回任务
    P->>Q: 各池独立召回候选
    Q-->>P: 候选题集
    P-->>R: 各池返回采样
    R->>R: 去重 + 场景覆盖 + 排序
    R-->>U: 题目列表
    U->>R: 答题 + 拍照提交
    R->>R: 批量 LLM 分析
    R->>DB: 更新 mastery/写错题/更新 review_queue
    R-->>U: 解析页 + 下一步推荐
